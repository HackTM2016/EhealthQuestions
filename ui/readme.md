HackTM front-end project
---
If you get permissions error, use `sudo` before `npm`

**Install project**
```bash
npm install
```

**Run project**
```bash
npm run
```

It opens a new tab with http://localhost:8000. Any change to `js|jsx|css|html` files will reload the tab.
